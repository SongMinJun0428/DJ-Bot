## 1. 봇 초기 설정 (Discord Developer Portal)

봇이 정상적으로 작동하려면 디스코드 개발자 포털에서 다음과 같은 설정이 필요합니다.

1.  **Privileged Gateway Intents**:
    *   [Discord Developer Portal](https://discord.com/developers/applications)에 접속하여 봇을 선택합니다.
    *   **Bot** 메뉴에서 아래 3가지 항목을 모두 **ON**으로 설정합니다.
        *   Presence Intent
        *   Server Members Intent
        *   **Message Content Intent** (필수!)
2.  ** OAuth2 권한**: 봇 초대 시 `bot` 스코프와 `applications.commands` 권한을 요청하세요.

## 2. 서버 및 채널 설정

봇을 초대하신 후, 전용 채널을 만들기 위해 다음 명령어를 사용하세요.

*   **/설정 (`/setup`)**: `DJ 봇 음악 채널`이라는 이름의 전용 채널을 자동 생성합니다.
*   **음악 전용 채널 기능**:
    *   **상시 대시보드**: 채널 상단에 인기차트, 최신곡, 검색 버튼이 고정됩니다.
    *   **즉시 재생**: 제목을 입력하거나 주소를 붙여넣고 엔터를 치면 즉시 재생됩니다.
    *   **파일 드래그 & 드랍**: MP3 파일을 채널에 올리는 것만으로 즉시 예약/재생이 가능합니다.

## 3. 주요 명령어 및 인터페이스 안내

*   **/play [제목/링크/파일]**: 음악을 검색하거나 재생합니다.
*   **/skip**: 현재 곡을 건너뛰고 다음 곡으로 이동합니다. (마지막 곡일 경우 자동 퇴장)
*   **/queue**: 전체 대기열 목록을 임베드로 확인합니다.
*   **/playlist**: 나만의 전용 플레이리스트를 서버 DB에 저장하고 불러옵니다.
*   **전용 버튼 컨트롤러**:
    *   `⏯️ 정지/재개`: 음악을 일시 정지하거나 다시 재생합니다.
    *   `🔁 반복설정`: 현재 곡 반복 재생을 제어합니다.
    *   `⏹️ 종료/퇴장`: 재생을 완전히 중지하고 초기 대시보드 상태로 돌아옵니다.

## 4. 기술 사양 및 요구 사항

*   **Runtime**: Node.js **v22.12.0+** (최신 암호화 대응 필수)
*   **Engine**: Lavalink v4.0.0 (Shoukaku 및 Kazagumo 연동)
*   **Database**: Better-SQLite3 (안정적인 데이터 관리)
*   **UI Framework**: Discord.js v14.18.0+

## 5. 배포 및 호스팅 가이드

### Railway 배포 (추천)
- Node.js 버전을 22 이상으로 설정하고, 변수에 `DISCORD_TOKEN`을 입력하세요.

### PM2를 사용한 서버 실행
```bash
npm install                     # 패키지 설치
npm install -g pm2              # PM2 전역 설치
pm2 start index.js --name "dj"  # 서비스 실행
```
