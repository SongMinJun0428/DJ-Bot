# 🎧 프리미엄 오디오 스트리밍 DJ BOT (v4.2.0)

이 프로젝트는 최신 Lavalink 엔진을 기반으로 하며, 고품격 **Premium Gold & Onyx** 디자인을 적용한 디스코드 음악 봇입니다. (v4.2.0 통합 업데이트 버전)

---

## ✨ v4.2.0 프리미엄 통합 업데이트 (What's New)

- **💎 Premium Gold UI**: 호텔 라운지 및 프리미엄 오디오 브랜드의 감성을 담은 미니멀 & 럭셔리 디자인을 적용하였습니다.
- **📂 통합 라이브러리 허브**: 즐겨찾기와 개별 플레이리스트를 '곡 검색 창'과 동일한 **번호 선택 방식**으로 통일하여 최상의 조작 편의성을 제공합니다.
- **🚀 2단 통합 컨트롤 패널**: 대기열 관리, 반복 재생뿐만 아니라 '매드무비 BGM', '인기 차트', '최신곡' 등 총 10가지의 퀵 기능을 대시보드에서 바로 제어할 수 있습니다.
- **🎧 자동 재생(Autoplay) v2**: 대기열이 비었을 때 유튜브 및 유튜브 뮤직 엔진을 동시 가동하여 끊김 없는 음악 추천 기능을 제공합니다.
- **🛠️ 엔진 최적화**: 채널 정보 실시간 동기화 및 메시지 자동 정리 기능을 통해 항상 깔끔한 음악 채널 상태를 유지합니다.

## 1. 초기 연동 가이드

봇의 완벽한 작동을 위해 디스코드 개발자 포털에서 아래 설정이 반드시 필요합니다.

1. **Privileged Gateway Intents**:
   * **Message Content Intent**: ON (사용자 메시지 인식 필수)
   * **Presence Intent**: ON
   * **Server Members Intent**: ON
2. **봇 초대 권한**: `Administrator` 또는 `Manage Channels`, `Read Messages`, `Send Messages`, `Connect`, `Speak` 권한이 필요합니다.

---

## 2. 서버 및 채널 설정

봇을 서버에 초대한 후, 단 한 번의 명령어로 전용 음악 룸을 생성하세요.

*   **/setup (`/설정`)**: 고품격 대시보드가 탑재된 `DJ 봇 음악 채널`이 자동 생성됩니다.
*   **전용 채널 활용법**:
    *   **즉시 검색**: 채널에 곡 제목이나 URL을 입력하고 엔터를 치면 검색 결과가 나타납니다.
    *   **10가지 선택 UI**: 검색 결과 중 원하는 번호를 메뉴에서 골라 바로 재생을 시작하세요.
    *   **파일 재생**: MP3 등 음악 파일을 채널에 업로드하여 즉시 재생 목록에 추가할 수 있습니다.

## 3. 프리미엄 대시보드 버튼 안내

| 버튼 | 기능 설명 |
| :--- | :--- |
| **인기차트/빌보드** | 오늘의 인기 음악을 10가지 선택지로 바로 제공 |
| **최신곡/매드무비** | 최신 트렌드 및 매드무비용 하이라이트 BGM 검색 |
| **📂 플레이리스트** | 즐겨찾기 및 저장된 모든 플레이리스트 통합 관리 및 재생 |
| **❤️ 보관함 저장** | 현재 듣고 있는 곡을 즉시 내 즐겨찾기에 추가 |
| **🎧 자동재생** | 대기열이 비었을 때 추천 곡을 자동으로 재생 (On/Off) |

## 4. 기술 사양 (Technical Stack)

* **Runtime**: Node.js **v22.12.0+**
* **Audio Engine**: Lavalink v4.0.0 (via Kazagumo & Shoukaku)
* **Storage**: Better-SQLite3 (고성능 로컬 데이터베이스)
* **UI**: Discord.js v14.18.0 (Embeds & ActionRows)
